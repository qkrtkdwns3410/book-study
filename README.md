## 🛠️ 스터디 규칙 (Rule)
1. **직접 커밋**: 본인의 Fork 레포가 아닌, 이 **조직(Org) 레포**에 직접 접속하여 파일을 업로드합니다.
2. **개인 폴더**: 각 책 폴더 내 `members/[본인아이디]` 폴더 안에서 자유롭게 작업합니다.
3. **스터디 방식**은 책 마다 상이하며, 최대한 책의 전달하려는 지식에 맞는 스터디법을 스터디원들끼리 연구하여 결정하고 수정하는 등의 방향으로 **유연한 규칙**을 적용합니다.
4. 각 하위 레포(책 항목)의 `README.md`에는 **책 선정의 이유**와 **스터디 방법론**, 각 멤버들의 **공부 결과물**의 링크가 존재합니다.
5. 스터디는 사내 스터디룸에서 퇴근 이후 진행되며, **약 1시간 내외**로 끝낼 것을 목표로 합니다.
6. 스터디는 특별한 사유가 없다면 **매주** 진행됩니다.
