# 가상 면접 사례로 배우는 대규모 시스템 설계 기초

## 개요
AI의 비약적인 발전으로 단순 구현은 기계가 대체하는 시대가 되었기에, 이제 개발자에게 필요한 핵심 역량은 코드 한 줄을 적는 기술보다 시스템 전체의 흐름을 조망하고 최적의 인프라를 선택하는 **'설계 사고력'**이라고 판단했다. 따라서 구현 너머의 거시적인 구조를 이해하고, 트래픽의 유입부터 글로벌 데이터 동기화까지의 전 과정을 논리적으로 설계할 수 있는 안목을 기르기 위해 이 책을 선택하였다.

## 참여자
* 이동석
* 박상준
* 홍준기
* 이지훈

## 스터티 방법
* 주 마다 1장 씩 읽고, 책의 내용에서 스스로 질문점을 만드는 등의 추가 조사 내용을 정리하여 md파일로 업로드
* 매주 정해진 시간에 약 1시간 내외로 자신들의 조사 내용에 대한 발표 및 토론

## 비고
총 15장 -> 15주 예정 (3 ~ 4달)
